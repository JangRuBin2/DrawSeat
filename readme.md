# DrawSeat

교실 자리 배정 & 번호 추첨 웹 프로그램

🔗 **[https://jangrubin2.github.io/DrawSeat/](https://jangrubin2.github.io/DrawSeat/)**

---

## 기능

### 🪑 자리 뽑기

| 단계 | 내용 |
|------|------|
| 1. 학생 설정 | 1번~n번 범위 입력, 결번(전학·결석 등) 선택 |
| 2. 좌석 배치 편집 | 10×8 격자에서 셀 클릭으로 자리 추가/삭제, 드래그로 이동 |
| 3. 자리 배정 | 드래그&드롭 또는 클릭으로 직접 배정 후 나머지 랜덤 배정 |
| 4. 결과 확인 | 드래그&드롭으로 자리 교환, 다시 뽑기, 인쇄 |

### 🎰 미니게임 — 슬롯머신

- 1번~n번 학생 중 결번 제외하고 추첨
- 특정 번호에 **추가 투표권** 부여 가능 (가중치 설정)
- 화려한 슬롯머신 애니메이션 + 🎉 컨페티 이펙트
- 회차별 추첨 기록 표시
